Creates an Address for a Key

#### Input
* ##### TF 
0 = root; 1 = system; 2 = constant; 3 = zero
* ##### RN 
Name of the region
* ##### MF 
Optional middle folders between region and file
* ##### FN 
Name of the file

#### Output
