Timing parameters of the simulation

#### Input
* ##### W 
Weather instance.
* ##### SD 
Index of the day of the year to start the simulation on
* ##### ST 
Starting hour of the simulation
* ##### D 
Number of hours to simulate

#### Output
* ##### T
Simulation timing settings
